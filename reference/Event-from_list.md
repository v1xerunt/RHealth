# from_list: Create Event from row

Creates a new Event object from a named list.

## Arguments

- row:

  Named list of attributes.

- event_type_col:

  Name of column to use as event_type.

- timestamp_col:

  Name of column to use as timestamp.

## Value

An Event R6 object.
