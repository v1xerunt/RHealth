# List Supported Medical Code Systems

Returns a character vector of the internal code systems currently
supported by the package.

## Usage

``` r
supported_inner()
```

## Value

A character vector of supported code system identifiers.

## Examples

``` r
supported_inner()
#> [1] "ICD9CM"    "ICD9PROC"  "ICD10CM"   "ICD10PROC" "ATC"       "CCSCM"    
#> [7] "CCSPROC"   "NDC"      
```
